# The Worlds — episode documents

One document per episode. An episode doc is the source of truth for what the
episode *is*: its arc, its acts, what the narration is allowed to say, what
data it draws from, and what machinery it needs. Docs for shipped episodes
describe what exists; docs for future episodes are the design spec the build
follows.

The one rule that makes this series what it is, and that every episode doc
inherits:

> **Nothing is canned.** Every fact the narrator speaks is read off the
> simulation for the viewer's own seed — this star's mass, this planet's
> temperature, this creature's leg length — so the narration is true by
> construction. Prose is written as seeded variants, so two universes never
> tell their story in the same words. If the simulation can't produce a fact,
> the narrator doesn't say it.
